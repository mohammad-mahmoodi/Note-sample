
## Notes App Compose 📒

- The Notes App is a simple and intuitive application designed for creating, managing, and organizing your notes seamlessly with reminder 

### Features 📱
- Create and Edit Notes
- Delete Notes
- Share the Notes

### Concepts Used ⚙️

- Android Jetpack
- MVVM+Clean Architecture
- Navigation Components
- Dagger Hilt
- RoomDB
- Coroutines
